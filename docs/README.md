# 文档目录

本目录用于存放“敏捷和DevOps成熟度”项目的说明文档。

## 文件说明

- `assessment-model.md`
  - 说明评估维度、评分方式、等级判定和报告输出逻辑
- `user-guide.md`
  - 说明系统的使用流程、问卷操作方式与报告导出方式

## 适用对象

- 咨询顾问
- 交付经理
- 产品负责人
- 项目管理人员
- 需要做 Agile / DevOps 现状诊断的客户团队

## 文档维护建议

- 当题库维度或题目数量调整时，同步更新 `assessment-model.md`
- 当页面交互或导出方式调整时，同步更新 `user-guide.md`
- 当项目脚本、端口或依赖有变化时，同步更新根目录 `README.md`
