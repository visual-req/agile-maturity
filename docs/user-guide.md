# 使用说明

## 1. 创建评估

进入 `评估配置` 页面后，填写以下信息：

- 客户名称
- 团队名称
- 行业
- 团队规模
- 评估人
- 评估日期
- 评估范围

填写完成后，点击 `开始问卷评估`。

## 2. 完成问卷

进入问卷评估页后：

- 左侧为评估维度导航
- 中间为答题区域
- 支持滚动条拖拽浏览长问卷
- 点击左侧维度可锚点定位到对应题组

### 问卷页交互说明

- 顶部操作区支持折叠 / 展开
- 每个维度顶部说明卡片支持折叠 / 展开
- 折叠某个维度后，会隐藏该维度下的所有题目
- 左侧导航点击某个维度时，会自动展开对应维度

### 评分说明

每道题按 `1-5 分` 评分：

- `1 分`：基本缺失
- `2 分`：局部实践
- `3 分`：形成常规机制
- `4 分`：稳定落地并持续优化
- `5 分`：组织级成熟能力

每道题还可以补充备注，用于记录证据、现状或观察结论。

## 3. 生成报告

当所有题目完成评分后，点击 `生成报告`。

系统将自动输出：

- 综合得分
- Agile 得分
- DevOps 得分
- 综合等级
- 维度分析
- 优势与现状
- 关键风险
- 改进建议

## 4. 导出报告

结果报告页支持以下导出方式：

- `导出摘要`
  - 导出 TXT 文本摘要
- `导出 PDF`
  - 按当前报告页面生成 PDF 文件
- `导出 Word`
  - 导出可直接用 Word 打开的文档文件
- `打印视图`
  - 使用浏览器打印功能输出报告

## 5. 建议使用方式

- 适合在访谈、调研、工作坊或交付诊断场景中使用
- 建议由顾问或交付负责人主导评分
- 建议在评分时同步记录证据，便于后续生成可解释报告
- 建议在报告输出后，结合短期 / 中期 / 长期建议形成行动计划
